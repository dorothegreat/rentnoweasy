Project: RentNowEasy

Purpose:
A platform connecting tenants directly with landlords.

Tech Stack:
- Next.js (App Router)
- TypeScript
- Supabase (Auth + Database + Storage)
- React Hook Form
- Zod validation
- Tailwind CSS
- Deployed on Vercel

Key Features:
- Property listing system
- Multi-step landlord listing form
- Tenant property search
- Landlord contact unlock system
- Subscription system (5 landlord contacts per subscription)

Important Rules:
- Do not break Vercel build
- Maintain TypeScript safety
- Supabase handles authentication
- Images stored in Supabase storage
